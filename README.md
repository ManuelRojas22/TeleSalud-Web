# 🏥 TeleSalud

**TeleSalud** es una interfaz web de una plataforma de telemedicina diseñada para organizar servicios de atención médica a través de un sistema digital.

El proyecto presenta la estructura de un sistema médico que permite gestionar información y acceder a diferentes módulos dentro de una plataforma de salud.

---

# 📌 Descripción

Se desarrolló una aplicación web que simula una plataforma de telemedicina.
El sistema incluye varias secciones que representan funcionalidades comunes en plataformas de salud digital.

Entre ellas se encuentran el panel principal del usuario, la gestión de citas médicas, el historial de pacientes y un espacio para videollamadas.

---

# ⚙️ Funcionalidades implementadas

El sistema incluye los siguientes módulos:

* **Página principal**
* **Inicio de sesión**
* **Dashboard (panel principal)**
* **Agenda de citas médicas**
* **Historial médico**
* **Interfaz de videollamada**

Cada sección permite navegar dentro del sistema simulando el funcionamiento de una plataforma de atención médica digital.

# 🎯 Objetivo del proyecto

El objetivo de TeleSalud es representar la estructura de una plataforma de telemedicina y demostrar la organización de un sistema web con múltiples módulos dentro de una misma aplicación web.
